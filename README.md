# Path of Exile 2 - Build Planners



## Get Started

This repository allows you to easily download and use all of the build files with minimal setup or adjustments.



### 1. Download the repository.

Click the green *Code* button and select *Download ZIP*.

### 2. Unzip the repository.

Navigate to where you downloaded the ZIP file. Right-click and *Extract All*.

### 3. Copy the *BuildPlanner* folder into your *Path of Exile 2* directory.

On (Windows) this is typically C:\Users\<user>\Documents\My Games\Path of Exile 2\

If you are uncertain, launch *Path of Exile 2*, open *Options*, select the *Game* tab, and click the folder icon next to the *Item Filter* option in the *Guidance* section.

### 4. Select your build.

Once on your character, open the *Passive Tree (P)* and select the build from the drop-down in the top left.
